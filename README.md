# Basic Website Practice #1 

## Requirements
Browser: Edge, Chrome, CocCoc, Firefox, Safari.
`nodejs` and `npm`.

## Build and run
Install all dependencies:
```shell script
$ npm install
```
Build for debugging:
```shell script
$ npm run build
```
Start development server:
```shell script
$ npm start
```
